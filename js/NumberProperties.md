extension from [[JavascriptNumber|javascript number]]
can not be used on a variable
`Number.xxx` is the only way to call them
`.MAX_VALUE` and `.MIN_VALUE` calls the largest / lowest possible number
`.MAX_SAFE_INTERGER` and `.MIN_SAFE_INTERGER` calls the largest / lowest safe interger
`.POSITIVE_INFINITY` and `.NEGATIVE_INFINITY` is essentially 1 / 0 and -1 / 0
`.NaN` is essentially num x not num
