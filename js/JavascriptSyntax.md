[[Syntax|syntax]]
case-sensitive
has `;` at the end of executable commands
can line break
ignores multiple spaces

[[FixedValues|fixed values]] 
numbers eg `10` or `11.5`
strings eg `'John Doe'` or `"Jane Doe"`

[[VariableValues|variable values]]
created using `let`, `var` or `const` 
must be identfied with unique names

[[Identifiers|identifiers]]
must start with a letter,`_`, or `$`
can contain digits but not first character
can not be reserved keywords
case-sensitive (same as )

[[Operators|operators]]
`=` assign value to given keywords
`+`, `-`, `*`, `/` to compute values

[[JoinWords|join words]]
underscores
upper camel case
lower camel case

statements: 
composed of `Values`, `Operators`, `Expressions`, `Keywords`, `Comments`

blocks:
code blocks are defined in 
```javascript
{
    curly brackets
}
```

[[Keywords|keywords]]
var
let
const
if
switch
for
function
return
try

[[Comments|comments]]
`//` is for single line
`/*` and `*/` are for multiple lines
