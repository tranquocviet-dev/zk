extension from [[JavascriptIfConditions|javascript if conditions]]

when activated, the command does different pre-determined things based on the situation

```javascript
switch (command){
	case 1:
		something = here;
	case 2:
		something_else = here;
	default:
		def = optional;
		this_is = fallback;
}
```
