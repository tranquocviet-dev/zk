extension from [[Javascript|javascript base]]
loops execute a block of code multiple times

code declared within loops can only be seen inside the loop

types of loops:
[[js/For|for]]
[[js/While|while]]
[[js/DoWhile|do while]]

[[js/Break|break]] disrupts the loop immediately
[[js/Continue|continue]] skips the following command of a loop
