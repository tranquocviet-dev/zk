extension from [[StringMethod|string method]]
`repeat()` repeats a string n times
`replace()` replaces a value in string with another, only the first match unless used /content/g
`replace all` replaces all values in string with another
`split()` splits the string into parts on the value selected, if used no value the output will be an array
