extension from [[JavascriptString|javascript string]]

[[IndexOf|indexOf]]: returns first occurance of a string, '-1' means none found
[[LastIndexOf|lastIndexOf]]: returns last occurance of a string, '-1' means none found
^ both accept a number after the string to specify start position

[[js/Search|search]]: search for a query and return locaton

`match()` returns an array of matching results between a string and a query
`matchAll()` returns all matching results between a string and a query
[[Includes|includes]] finds whether the string contains a value (returns in [[Boolean|boolean]])

`startWith()` and `endWith()` checks whether the string starts or begins with a value (returns in [[Boolean|boolean]])
